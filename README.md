# IT Asset Search App (Power Apps)

Internal Power Apps canvas application for searching and viewing IT asset records (devices, peripherals, and assigned users/locations).  
Built to improve inventory visibility and reduce time spent manually locating asset information.

## Features

- Asset lookup with fast search + filtering
- Detail view for asset metadata
- Search logging for auditing/usage insights

## Tech

- Microsoft Power Apps (Canvas App)
- Power Platform CLI (source unpacked for version control)

## Notes

This repository contains the unpacked source files generated from the `.msapp` using Power Platform CLI for better diffing and collaboration.

Connections are placeholders (Excel/OneDrive). Importers will need to rebind connections in their own environment.
